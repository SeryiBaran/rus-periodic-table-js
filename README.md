# periodic-vue

Периодическая таблица в стандартном для России коротком формате на Vue 3.

Данные взяты из проекта [Periodic-Table-JSON](https://github.com/Bowserinator/Periodic-Table-JSON) и на всякий случай поставляются вместе с исходниками в директории `@/src/data`.

[![Скриншот](./screenshot.png)](./screenshot.png)

Использован `PNPM` и `VITE+ v0.1.11`.
